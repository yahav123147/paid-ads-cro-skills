# 🎬 סקיל עריכת וידאו ל-Claude Code

קיבלת סקיל שהופך סרטונים ארוכים לקליפים אנכיים מוכנים לרשת: חיתוך לרגעים החזקים, כתוביות קריוקי בעברית (מילה מודגשת בזמן שהיא נאמרת), **זום שעוקב אחרי הפנים שלך**, טיזרים עם מוזיקה, צנזור מילים שפייסבוק לא אוהב, וניקוי רעש רקע.

## מה צריך
- **Mac עם שבב Apple Silicon** (M1/M2/M3 ומעלה). התמלול משתמש ב-`mlx` של אפל.
- **Claude Code** מותקן.
- `Node.js` ו-`python3` (בדרך כלל כבר קיימים במק).

## התקנה (דקה)
1. לפתוח (unzip) את התיקייה `video-editing`.
2. להעביר אותה לתיקיית הסקילים של Claude Code, כך שיהיה לך:
   ```
   ~/.claude/skills/video-editing/SKILL.md
   ```
   אם התיקייה `~/.claude/skills/` לא קיימת, פשוט ליצור אותה.
3. זהו. Claude Code יטען את הסקיל אוטומטית.

## איך משתמשים
פשוט תבקש מ-Claude Code בשפה חופשית, למשל:
> "יש לי סרטון על שולחן העבודה, תחתוך לי אותו לקליפים עם כתוביות וזום שעוקב אחריי"

הסקיל ייטען לבד. בפעם הראשונה הוא יריץ `bash scripts/setup.sh` שמתקין את הכלים (`ffmpeg`, `mlx-whisper`, `opencv`) - זה קורה פעם אחת ולוקח כמה דקות.

## מה יש בפנים
- `SKILL.md` - כל התהליך, הפרמטרים, והטריקים.
- `scripts/` - סקריפטים מוכנים (מעקב פנים, חיתוך-זום, כתוביות, טיזרים, ניקוי רעש).
- `models/` + `fonts/` - מודלים לזיהוי פנים ופונט, **מצורפים** (אין הורדות).

## הערות
- הסקיל בנוי ל-**macOS Apple Silicon**. על מערכות אחרות חלק התמלול צריך התאמה.
- הפונט `Arial` הוא פונט מערכת (קיים בכל מק/חלונות). אם בא לך פונט אחר, מחליפים ב-`fonts/`.
- רשימת המילים לצנזור נמצאת ב-`scripts/captions_hz.py` - אפשר להוסיף/להסיר לפי הצורך.
