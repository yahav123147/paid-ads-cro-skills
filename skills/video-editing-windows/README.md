# 🎬 סקיל עריכת וידאו בעברית ל-Claude Code (חוצה-פלטפורמה)

קיבלת סקיל שהופך סרטונים ארוכים לקליפים אנכיים מוכנים לרשת: חיתוך לרגעים החזקים, כתוביות קריוקי בעברית (מילה מודגשת בזמן שהיא נאמרת), **זום שעוקב אחרי הפנים שלך**, טיזרים עם מוזיקה, צנזור מילים שפייסבוק לא אוהב, וניקוי רעש רקע.

**רץ על Windows, macOS ו-Linux.**

## מה צריך
- **Windows / macOS / Linux.** על ווינדוס: Python 3.9+ מ-[python.org](https://python.org) (בהתקנה לסמן "Add Python to PATH"). על מק/לינוקס: `python3` (בדרך כלל כבר קיים).
- **Claude Code** מותקן.
- אין צורך להתקין ffmpeg בנפרד — הוא מגיע מובנה.

## התקנה
1. לפתוח (unzip) את התיקייה `video-editing-windows`.
2. להעביר אותה לתיקיית הסקילים של Claude Code, כך שיהיה לך:
   ```
   ~/.claude/skills/video-editing-windows/SKILL.md
   ```
   (בווינדוס: `C:\Users\<שם>\.claude\skills\video-editing-windows\SKILL.md`). אם התיקייה `.claude/skills/` לא קיימת, פשוט ליצור אותה.
3. זהו. Claude Code יטען את הסקיל אוטומטית.

## איך משתמשים
פשוט תבקש מ-Claude Code בשפה חופשית, למשל:
> "יש לי סרטון בתיקיית ההורדות, תחתוך לי אותו לקליפים עם כתוביות וזום שעוקב אחריי"

הסקיל ייטען לבד. בפעם הראשונה הוא יריץ `python scripts/setup.py` שמתקין את הכלים (`faster-whisper`, `opencv`, `pillow`, `imageio-ffmpeg`) - זה קורה פעם אחת ולוקח כמה דקות. התמלול מוריד בפעם הראשונה מודל שפה (~1.5GB) ושומר אותו במטמון.

## מה יש בפנים
- `SKILL.md` - כל התהליך, הפרמטרים, והטריקים (כולל גוצ'ות ווינדוס).
- `scripts/` - סקריפטים מוכנים (תמלול, מעקב פנים, חיתוך-זום, כתוביות, טיזרים, ניקוי רעש) - הכל בפייתון, חוצה-פלטפורמה.
- `models/` + `fonts/` - מודלים לזיהוי פנים ופונט, **מצורפים** (אין הורדות).

## הערות
- התמלול הוא `faster-whisper` - רץ על כל מעבד, ואם יש כרטיס מסך NVIDIA הוא ינוצל אוטומטית ויהיה הרבה יותר מהיר.
- הפונט `Arial Bold` מצורף. אם בא לך פונט אחר, מחליפים ב-`fonts/`.
- רשימת המילים לצנזור נמצאת ב-`scripts/captions_hz.py` - אפשר להוסיף/להסיר לפי הצורך.
- למשתמשי מק על שבב Apple שרוצים תמלול מהיר יותר עם MLX - יש גם גרסת מק ייעודית ("עריכת וידאו בעברית").
